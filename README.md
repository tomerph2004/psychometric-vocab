# 📚 אפליקציית לימוד אוצר מילים פסיכומטרי

אפליקציית ווב ללימוד אוצר מילים לקראת הבחינה הפסיכומטרית (חשיבה מילולית).

## תכונות

- 📇 **כרטיסיות** — למידה עם חזרה מרווחת (Spaced Repetition)
- 🎯 **חידון** — בחן את עצמך עם 4 אפשרויות
- 📝 **חזרה על שגיאות** — תרגול ממוקד של מילים שטעית בהן
- 📊 **דשבורד סטטיסטיקות** — מעקב אחר ההתקדמות
- 🌙 **מצב כהה/בהיר**
- 📱 **רספונסיבי** — עובד גם בטלפון

## קטגוריות

| קטגוריה | דוגמאות |
|---|---|
| פעלים נדירים | לחונן, לחדול, להתבצר |
| שמות עצם מופשטים | תבונה, חירות, ענווה |
| תארים ספרותיים | נעלה, ארעי, דליל |
| ביטויים ומילות קישור | יתר על כן, אף על פי, אולם |
| משפט ופילוסופיה | עילה, הכרעה, חפות |
| מדע ואמנות | השערה, תגלית, פרשנות |
| נרדפות ומשמעות כפולה | בהיר, כבד, חד |
